# Static Analysis Tools

## Facebook Infer

If you have a machine running Linux or macOS, try this tool out on some
of the code you've seen this week.

You can [download it from GitHub][inf].

## Flawfinder

Visit the [Flawfinder website][flaw], consult its user manual, then download
the tool. Try running it on some of the C programs from this and previous
sessions. What problems do you see?

Note: you will need to run this on a system that has Python installed.


[inf]: https://github.com/facebook/infer
[flaw]: https://dwheeler.com/flawfinder/
